# HyperPoll
Virtual 8KHz polling rate enabler for standard 1KHz mice – reaches 1500-4000Hz on tested models. By Valerois.


# Valerois Pollex 🖱️⚡

**Valerois Pollex**, standart 1.000 Hz (1K) USB farelerin polling rate'ini yazılımsal olarak artıran sanal bir **8KHz mouse** çözümüdür.  
Gerçek 8K donanımı olmayan farelerde bile yapılan optimizasyonlarla **1500-4000 Hz** arası kararlı polling elde edilmiştir.

> ⚠️ **Deneysel projedir.** Donanım garantisini etkileyebilir, tüm risk kullanıcıya aittir.

---

## 🔍 Nasıl Çalışır?

Yazılım, USB tanımlayıcılarını ve polling aralıklarını sistem seviyesinde değiştirerek standart bir fareyi yüksek polling rate’e çıkarır. Hedef, **gerçek zamanlı oyun ve hassas hareket** senaryolarında gecikmeyi azaltmaktır.

- **Desteklenen işletim sistemi:** Windows 10/11 (önerilir)
- **Başarılı test aralığı:** 1500 Hz – 4000 Hz (cihaza ve anakart USB denetleyicisine bağlı)
- **Teorik maksimum:** 8000 Hz (özel donanım veya uyumlu hub’lar ile)

---

## 📦 Kurulum

1. Virtual8K.exe dosyasını çalıştırın.
2. Ayarlar kısmından kendiniz test ederek ayarlarınızı bulun.
---

## 🛠️ Sorun Giderme

- Fare imleci titriyorsa polling rate’i düşürün.
- `Error: Access Denied` → Uygulamayı yönetici olarak çalıştırdığınızdan emin olun.
- Fare hiç algılanmıyorsa, farklı bir USB portu (tercihen doğrudan anakart USB 3.0) deneyin.
